#TagCloud

![mahua](http://github.com/yumemor/TagCloud/raw/master/top_logo.gif)

* 基于官网文件修改swf文件，方便配置各种`透明`标签云。

* 使用：
    
    拷贝tagcloud.swf文件到项目，在html页面添加如下代码

```html
<embed type="application/x-shockwave-flash" src="tagcloud.swf ?>" width="250" height="250" id="tagcloudflash" name="tagcloudflash" bgcolor="#ffffff" quality="high" wmode="transparent" allowscriptaccess="always" flashvars="tcolor=0x666666&tcolor2=0x666666&hicolor=0x666666&tspeed=100&distr=true&mode=tags&tagclou=<tags>a标签</tags>">
```
    
    >具体使用以及参数详情列表，请访问[官网](http://www.tagcloud-generator.com)。
